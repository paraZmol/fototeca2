# Directrices de Interfaz
1. MOBILE-FIRST: Diseñar primero para pantallas pequeñas. El menú lateral se convierte en un "hamburguesa" o un modal en móviles.
2. GRID FLEXIBLE: La galería de "estantes" debe ser de 1 columna en móvil, 2 en tablet y 3 o 4 en desktop.
3. IMÁGENES: Implementar object-fit: cover para que las fotos no se deformen y usar lazy loading nativo para ahorrar ancho de banda.
4. ACCESIBILIDAD: Mantener el alto contraste (textos claros sobre fondos oscuros) para facilitar la lectura.
5. INTERACCIÓN: Los elementos clickeables deben tener un tamaño mínimo de 44x44px en móviles.
