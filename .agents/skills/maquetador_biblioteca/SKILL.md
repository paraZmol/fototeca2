# Instrucciones de Diseño Visual
1. La interfaz pública debe evocar una biblioteca física: utilizar tonos madera (nogal, caoba) y fondos de alto contraste (gris grafito o sepia oscuro).
2. La galería debe estructurarse usando CSS Grid, simulando cuadros apoyados en estantes.
3. El panel de administración debe tener un modo oscuro (dark mode) nativo obligatorio para mantener un entorno de alto contraste y confort visual durante el desarrollo.
4. Genera solo HTML semántico y CSS puro estructurado. No uses librerías UI pesadas.
5. Idioma UI: Todo el texto visible para el usuario (etiquetas, botones, menús) debe ser en español latinoamericano.
6. Sidebar (Izquierda): Categorías geográficas (Provincias como Huaraz, Yungay, Recuay) y sus Subcategorías (Barrios, Calles, etc.) .Filtros Superiores (Especiales): Pestañas de distribución temporal ("Antes del Terremoto", "Terremoto", "Siglo XXI").
